# 🦌数据结构标准

## ⚙通用标准
返回json的数据结构:
```javascript
{
    // 0代表有错误,1代表成功
    "code":1,
    "msg": "提示信息",
    // 如果是列表数据就是"数组", 如详情就是"对象"
    "data": [],
    // 如果是列表数据, 此处表示列表信息的总数, 前端用来实现分页.
    "dataCount": 100
}
```

## 📈图表类

```javascript
{
    // 0代表有错误,1代表成功
    "code":1,
    "msg": "提示信息",
    // 注意: 数组中可以只有一个对象元素 
    // 请确保data是个数组
    "data":[
        // label: xxx/yyy是每一个线条的显示标题
        // x和y的名字可根据业务动态识别, 请尽量使用"x/y"
        {
            "label":"xxx",
            "data":[{"x":109, "y":"2020-01-01 10:32:11"}]
        },
        {
            "label":"yyy",
            "data":[{"x":44, "y":"2020-01-01 10:32:11"}]
        }
    ]
}
```
